Atlas Communication

Basic Interaction

  • • Greetings: Say hello to Atlas.
  • • Goodbye: Let Atlas know when you are finished.
  • • Out-Of-Scope: Used when a request falls outside Atlas's current abilities.
  • • Timer: "Set a [Duration] timer." You can also say "Set a [TimerName] timer for [Duration]".
  • • Weather: "What's the weather in [City]?" You can also specify a [Country].

Plant Knowledge

  • • Sunlight: "How much sun does a [PlantName] need?".
  • • Watering Care: "Get watering information for [PlantName]".
  • • Plant Cycle: "What is the cycle for [PlantName]?" (e.g., Perennial).
  • • Edibility: "Is [PlantName] edible?".
  • • Search by Environment: "Find [Quantity] [indoor/outdoor] plants".

Garden Control System

  • • Move: "Move [Direction]" (left, right, up, down). Optionally add "[NumTiles]".
  • • Pick Up Tool: "Pick up the [Tool]" (watering can or hoe).
  • • Pick Up Seed: "Pick up [Seed] seeds" (carrot, lettuce, or tomato).
  • • Plant: "Plant [Seed]" (carrot, lettuce, or tomato).
  • • Water: "Water the current tile".
  • • Till: "Till the current tile".
  • • Fill Water: "Fill my watering can at the Resource Station".
  • • Restart: "Restart the game" to reset the state.
  • • Game Help: "Get game instructions" for help with the control system.
